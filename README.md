# 📝 To-Do List Application

تطبيق قائمة مهام مع تخزين محلي (Local Storage)

## ✨ الميزات

- ✅ **إضافة مهام جديدة** - أضف مهامك بسهولة
- ✅ **تحرير المهام** - عدّل نصوص المهام
- ✅ **حذف المهام** - احذف المهام التي لا تحتاجها
- ✅ **وضع علامة على المهام المنجزة** - أتمّ مهامك وضع عليها علامة
- ✅ **التصفية** - عرض الكل، النشطة، أو المنجزة فقط
- ✅ **التخزين المحلي** - تُحفظ البيانات تلقائياً في متصفحك
- ✅ **إحصائيات** - عدد المهام الكلي والمنجز
- ✅ **واجهة رسومية جميلة** - تصميم احترافي وسهل الاستخدام
- ✅ **متجاوب** - يعمل على جميع الأجهزة (Desktop, Tablet, Mobile)

## 🚀 كيفية الاستخدام

1. **استنساخ المستودع** أو تحميل الملفات
2. **فتح `index.html`** في المتصفح
3. **البدء بإضافة مهامك**

```bash
git clone https://github.com/khalidSAT/tourabi-khalid.git
cd tourabi-khalid
# ثم افتح index.html في المتصفح
```

## 📁 الملفات

- `index.html` - الهيكل الأساسي
- `styles.css` - التصاميم والألوان
- `script.js` - الوظائف والمنطق
- `README.md` - هذا الملف

## 💾 التخزين المحلي

جميع مهامك تُحفظ تلقائياً في `localStorage` في متصفحك:
- البيانات تبقى حتى بعد غلق المتصفح
- لا تحتاج لقاعدة بيانات خارجية
- البيانات محفوظة محلياً على جهازك

## 🎨 التخصيص

يمكنك تخصيص الألوان في `styles.css`:

```css
/* الألوان الرئيسية */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* الألوان الثانوية */
border-color: #667eea;
```

## 📊 البنية المعمارية

```
TodoApp
├── loadFromStorage() - تحميل البيانات
├── addTodo() - إضافة مهمة
├── deleteTodo() - حذف مهمة
├── toggleTodo() - وضع علامة على المهمة
├── editTodo() - تعديل المهمة
├── clearCompleted() - حذف المهام المنجزة
├── setFilter() - تصفية المهام
└── saveToStorage() - حفظ البيانات
```

## 🎯 أمثلة الاستخدام

### إضافة مهمة
1. اكتب المهمة في حقل الإدخال
2. اضغط زر "Add Task" أو اضغط Enter

### تحرير مهمة
1. اضغط زر "Edit" بجانب المهمة
2. أدخل النص الجديد
3. اضغط OK

### حذف مهمة
1. اضغط زر "Delete" بجانب المهمة

### وضع علامة على المهمة
1. اضغط على Checkbox بجانب المهمة

### التصفية
1. اضغط على أحد أزرار التصفية: All, Active, Completed
